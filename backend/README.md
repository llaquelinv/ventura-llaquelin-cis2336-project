# ArtConnect

ArtConnect is a front-end art gallery website designed to help artists
showcase their work and help visitors discover artwork and community events.

## Developer

Llaquelin Ventura  
University of Houston  
CIS 2336 Web Development

## Project Features

- Responsive homepage
- Artwork gallery with JavaScript filtering
- Events page with dynamic event details
- Artwork submission form with client-side validation
- FAQ accordion
- Embedded YouTube multimedia content
- References page containing image, multimedia, and development sources

## Technologies Used

- HTML
- CSS
- JavaScript
- YouTube embedded media

## Project Structure

- `frontend/index.html` — Homepage
- `frontend/pages/gallery.html` — Artwork gallery
- `frontend/pages/events.html` — Art events
- `frontend/pages/submission.html` — Artwork submission form
- `frontend/pages/faq.html` — Frequently asked questions
- `frontend/pages/references.html` — Project references
- `frontend/css/style.css` — Website styling
- `frontend/js/script.js` — Interactive functionality
- `frontend/images/` — Website images

## How to Run the Project

1. Download or clone the repository.
2. Open the `frontend` folder.
3. Open `index.html` in a web browser.
